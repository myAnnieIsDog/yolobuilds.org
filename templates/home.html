Applications

Accessory Structures New construction of private garages, barns, sheds, towers, tanks, retaining walls, fences and other non-habitable and non-occupiable structures. See below for Pool/Spa.
Commercial New construction of halls, restaurants, offices, retail, factories, warehouses and similar occupiable spaces.
Dwellings New construction of custom homes, track homes, manufactured homes, tiny homes, accessory dwelling units (ADU's) and similar habitable spaces.
Demolition Any building or structure (including pools) or partial demolition to prepare for a remodel.
Electrical For work covered by the CA Electrical Code including Fire alarms, solar, batteries, EVCS, backup generators, and motor loads (such as well pumps).
Existing Building For work covered by the California Existing Building Code including repair, alteration, remodel, reconstruction, moved structures, and additions.
Fire Work regulated by the CA Fire Code including construction permits, operational permits, and fire sprinkler systems. In some fire districts you will work directly with the fire district.
Flood For any development regulated by the Yolo County Flood Protection Ordinance.
Grading For any earthwork regulated by CA Building Code Appendix J.
Mechanical Work regulated by the CA Mechanical Code including HVAC.
Plumbing Work regulated by the CA Plumbing Code including water heaters and fuel gas piping.
Pool/Spa For any pool and/or spa, including private residential and public pools. Private Residential Pools are regulated by the Internation Swimming Pool and Spa Code and the California Swimming Pool Safety Act. Public Pools are regulated by CA Building Code Chapter 31B Public Pools and the Department of Public Health.
